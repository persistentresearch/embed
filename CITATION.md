# How to Cite Persistent Embed

If you use Persistent Embed in your research, please cite:

## Primary Citation

```bibtex
@misc{persistentresearch2026embed,
  title        = {Persistent Embed: Open-Source Multilingual Embeddings
                  with Indian Language Focus},
  author       = {Persistent Research},
  year         = {2026},
  publisher    = {Persistent Research},
  howpublished = {\url{https://github.com/persistent-research/embed}},
  note         = {Model weights available at
                  https://huggingface.co/persistent-research}
}
```

## Specific Model Versions

### persistent-embed-50m

```bibtex
@misc{persistentresearch2026embed50m,
  title        = {Persistent Embed 50M: Efficient Multilingual Embeddings},
  author       = {Persistent Research},
  year         = {2026},
  url          = {https://huggingface.co/persistent-research/embed-50m}
}
```
